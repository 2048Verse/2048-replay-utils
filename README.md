# 2048-replay-utils
2048 replay format utilities

## Build

```bash
npm i
npm run build
```

## Node Usage

```bash
npm link
cd your-project
npm link 2048-replay-utils
```
```js
const replayUtils = require("2048-replay-utils");
```

## Browser Usage

```html
<script type="module">
    import * as replayUtils from "./index.esm.js";
</script>
```
